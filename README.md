# 📌 Image Post App (Full Stack)

A full-stack web application that allows users to upload images with captions and view all posts in a responsive feed.

🚀 Features

📸 Upload image with caption
☁️ Store images using ImageKit cloud storage
🗄️ Save post data in MongoDB
📡 REST API built with Express.js
🖥️ Responsive UI using React.js and Tailwind CSS
🔄 Fetch and display all posts dynamically
🛠️ Tech Stack

Frontend:
React.js
Tailwind CSS
Axios

Backend:
Node.js
Express.js
Multer

Database:
MongoDB

Cloud Storage:
ImageKit

🔐 Environment Variables
Create a .env file inside the backend folder and add the following:
......
MONGODB_URI=your_mongodb_connection_string
IMAGEKIT_PRIVATE_KEY=your_private_key
......
